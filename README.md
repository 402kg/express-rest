# express-boilerplate

- Express
- Mongoose
- Socket.io
- Babel7

#### Running
```bash
docker-compose up -d
```

#### Environment Setup

Run services:
```
docker-compose -f ./docker-services.yml up -d
```

Run application:
```
yarn start
```

#### Environment variables
```
USER_EMAIL = [initial user email] | admin@root
USER_PASSWORD = [initial user password] | rootroot
```
